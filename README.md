# PDP TV P65CGA - Mini QlLED

Design de página de produto para TV P65CGA Mini QlLED gerado automaticamente do Figma.

## 📋 Estrutura

```
├── index.html              # Página principal do produto
├── css/
│   └── styles.css         # Estilos globais (Tailwind)
├── js/
│   ├── darkMode.js        # Gerenciador de dark mode
│   └── interactions.js    # Interações da página
└── components/
    ├── header.html        # Header responsivo
    ├── hero.html          # Seção hero
    ├── features.html      # Recursos principais
    ├── specs.html         # Especificações técnicas
    ├── gallery.html       # Galeria de imagens
    └── footer.html        # Footer
```

## ✨ Recursos

- ✅ **Responsivo** - Mobile-first design
- ✅ **Dark Mode** - Tema escuro automático
- ✅ **Acessibilidade** - WCAG 2.1 AA
- ✅ **Tailwind CSS** - Utility-first
- ✅ **TypeScript Ready** - Para migração futura
- ✅ **Otimizado** - SEO e performance

## 🚀 Início Rápido

```bash
# Instalar dependências (opcional, só se usar build tools)
npm install

# Abrir no navegador
open index.html

# Ou usar um servidor local
npx http-server
```

## 🎨 Customização

- Cores: Edite `css/styles.css`
- Dark Mode: Edite `js/darkMode.js`
- Componentes: Edite arquivos em `components/`
